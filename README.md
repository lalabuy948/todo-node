# ToDo app written in node.js

## Requirements

```sh
# go to the directory
cd todo-node

# install node packages
npm i
```

## Usage

```zsh
node todo.js new
```

```zsh
  todo <command>

    commands can be:

    new:      used to create a new todo
    get:      used to retrieve your todos
    complete: used to mark a todo as complete
    help:     used to print the usage guide
```
